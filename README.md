# Face Detection & Recognition

 Description
Built a real-time Face Detection and Recognition
system using Python, OpenCV and face-recognition
library achieving 100% recognition accuracy.
# Results
- ✅ Detected faces with green bounding boxes
- ✅ Recognized faces with name labels
- ✅ Achieved 100% accuracy on test images
- ✅ Works on any uploaded image

# Tools Used
- Python
- OpenCV
- face-recognition
- NumPy
- Google Colab

# How to Run
1. Open notebook in Google Colab
2. Run pip install cell first
3. Upload a known face image
4. Upload a test image
5. See recognition results with accuracy score
